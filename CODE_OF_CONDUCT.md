# Code of Conduct

This is a small, public map. The rules are small too.

- Be kind. Be plain. Assume good faith.
- Critique the work, not the person.
- No harassment, no spam, no bad-faith noise.
- Keep discussion about the loop: copy it, run it, break it, report honestly.

If someone is acting in bad faith, contact us through https://mushku.com.

We may remove comments, issues, or contributors that break these rules.
